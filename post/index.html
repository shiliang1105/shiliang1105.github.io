目录与文件操作

主键盘快捷键： control u 清除命令行一行 control d 删除光标后面的 control k 删除光标到行末 control a first letter control e end

Linux系统目录：

目录 作用
bin 存放二进制可执行文件
boot 存放开机启动文件
dev 存放设备文件
home 存放普通用户
etc 用户信息与系统配置文件passwd、group
lib 库文件libc.so.6（标准c库）
root 管理员宿主目录（家目录）

cp命令： cp filename dirname 复制文件到目录 cp filename1 filename2 复制文件1 并重命名为文件2 cp -a dirname1 dirname2 复制目录1及其下所有文件到目录2 cp -r dirname1 dirname2 递归复制目录1到目录2 这里-a 和-r 的差别在于，-a 是完全复制，文件权限，改动时间什么的也完全相同。

软链接与硬链接：

  • 软链接： 软链接就像windows下的快捷方式。 ln -s file file.s 为file创建一个软链接file.s。 软链接的大小就是文件路径。 如果用相对路径创建的软链接，在软链接移动之后就会失效，无法访问。绝对路径创建的不会失效。 用ls -al查看软链接的权限时,其权限就是软链接的权限，不是软链接指向的文件的权限。

  • 硬链接： ln file file.h 创建一个硬链接file.h。 ln file file.hard 再为file创建一个硬链接file.hard。此时，对于file文件有两个硬链接file.h和file.hard，无论改变哪个硬链接或者文件本身，这三个文件本身都同步变化。这三个文件的Inode是一样的，每个文件的Inode都是唯一的。

文件属性和用户用户组

chmod修改权限操作

  • 第一种，文字设定法 chmod [who] [+|-|=] [mode] filename 操作对象who可以是下述字母中的任一个或者它们的组合。
操作对象 含义
u 表示“用户（user）”，即文件或目录的所有者
g 表示“同组（group）用户”，即与文件所有者有相同组ID的所有用户
o 表示“其它（other）用户”
a 表示“所有（all）用户”，他是系统默认值

操作符可以是：

操作符 含义
+ 添加某个权限
- 取消某个权限
= 赋予给定权限并取消其他所有权限（如果有的话）
  • 第二种，数字设定法 chomd 操作码 filename 直接用操作码修改文件权限 r、w、x权限用数字表示为：r对应4，w对应2，x对应1 如果file文件的权限为-rw-rw-r–，则owner的权限为rw- = 4+2+0 = 6, group的权限为rw- = 4+2+0 = 6, others的权限为r– = 4+0+0 = 4；所以该文件的权限数字为664。 chomd 764 filename 可以将filename文件的权限改为-rwxrw-r–。

sudo adduser newusername 添加新用户 sudo chown username filename 修改文件所有者 su username 切换当前用户为username sudo addgroup groupname 添加新的用户组 sudo chgrp groupname filename 修改用户所属用户组 sudo chown username:groupname filename 同时修改文件所属用户和用户组 sudo deluser username 删除用户 sudo delgroup groupname 删除用户组

find命令

找文件：

  • -type 按文件类型搜索 d/p/s/c/b/l/(f:文件)
  • -name 按文件名搜索 例如：find ./ -name ‘*.jpg’
  • -maxdepth 指定搜索深度。要作为第一个参数出现。 例如：find -maxdepth 2 -name '*.jpg' 找2层（包含第一层）目录第文件
  • -size 按文件大小搜索，单位:b、c、w、k、M、G b表示块，1b = 512 byte = 0.5 k 例如：find ./ -size +20M -size -50M 两个size一个都不能少，文件大小单位对大小写敏感
  • -atime、mtime、ctime 天 amin、mmin、cmin 分钟
    按时间搜索 atime 表示最近访问时间 mtime 表示最近更改时间，指更改文件属性一类的 ctime 表示最近改动时间，指更改文件内容
  • -exec 将find搜索到的结果集执行某一指定命令 例如：find /usr/ -name '*tmp*' -exec ls -l {} \; 搜索包含“tmp”的文件集，然后将搜索到的文件放{}执行ls -l。
  • -ok 以交互的方式 将find搜索到的结果执行某一指令。 例如：find /usr/ -name '*tmp*' -ok ls -l {} \;
  • -xargs 将find搜索到的结果集执行某一指定命令 例如：find /usr/ -name '*tmp*' | xargs ls -l 搜索包含“tmp”的文件集，然后执行ls -l。 -exec与-xargs的区别：当结果集很大的时候，xargs会对结果进行分片映射，而exec不会，所以xargs性能会好一些。 xargs的缺陷为：它默认使用空格来分隔结果集，当文件名有空格时，会因为文件名被切割实效。解决方法：指定切分符号为null。使用-print0: find /usr/ -name '*tmp*' -print0 | xargs -print0 ls -l 第一个-print0指定结果集分割为null，第二个-print0指定xargs分隔为null。

创建名字带空格的文件方法：

  1. 文件名加引号：例如touch 'ab cd'
  2. 使用转义字符：例如touch 'ef\ gh'

grep命令

找文件内容： grep -r 'copy' ./ -n 在当前目录的文件中查找“copy”。-n参数可以显示行号。-r参数以递归的方式查找。 ps命令 监控后台进程工作情况。 ps 显示当前可以和用户交互的进程。 ps aux 显示当前系统所有用户的所有进程。 ps aux | grep 'cupsd' 检索包含’cupsd’的进程集。用grep搜索进程，如果只有一条结果，说明当前进程中没有你要检索的内容，显示的那一条是搜索本身占用的进程。

安装卸载软件

apt-get

sudo apt-get install softname 安装软件 sudo apt-get update 更新软件列表 sudo apt-get remove softname 卸载软件

使用安装包进行软件安装：

sudo dpkg -i xxx.deb 安装deb软件包命令 sudo dpkg -r xxx.deb 删除deb软件包命令 sudo dpkg -r --purge xxx.deb 连同配置文件一起删除命令 sudo dpkg -info xxx.deb 查看软件包信息命令 sudo dpkg -L xxx.deb 查看文件拷贝详情命令 sudo dpkg -l 查看系统已安装软件包信息命令 sudo dpkg-reconfigure xxx 重新配置软件包命令

源码安装

  1. 解压源代码包
  2. cd dir
  3. ./configure 检测文件是否缺失，创建Makefile，检测编译环境
  4. make 编译源码，生成库和可执行文件
  5. sudo make install 把库和可执行程序，安装到系统路径下
  6. sudo make disclean 删除和卸载软件

压缩与解压

gzip和bzip2

两者都是配合tar打包命令使用。 这两个压缩的缺陷是都只能对单个文件进行压缩，不能压缩目录也不能打包。

  • gzip

tar zcvf 要生成的压缩包名字 压缩材料 要生成的压缩包名字一般以.tar.gz结尾。 例如：tar zcvf spacefile.tar.gz ab\ cd ef\ gh 将ab\ cd文件和ef\ gh文件压缩在一起，压缩后的文件名为spacefile.tar.gz。 这个命令包含两步：

  1. gzip filename 压缩文件
  2. tar file... tarname 打包命令

gunzip zipfile 解压 所以 tar zcvf 是两条指令的结合版本 对 zcvf 进行解释: z:zip，压缩 c:create，创建 v:vision，显示压缩过程，可以去掉，直接用 zcf，但这样不显示压缩过程 f:file，文件 利用file filename命令可以查看文件来源。

  • bzip2

tar jcvf 要生成的压缩包名字 压缩材料 压缩命令 tar jxvf 压缩包名字 解压命令

rar和zip

  • rar

rar a -r newdir dir 打包，把dir压缩成newdir.dir，如果压缩材料里没有目录，-r参数可以省去。 例如：rar a -r dzip.rar Mydir ab\ cd ef\ gh unrar x newdir.rar 解压rar文件 sudo aptitude show softname 查看软件安装信息

  • zip

zip -r dir.zip dir 压缩 unzip dir.zip 解压

其他常用命令

who 查看当前在线上的用户情况 whoami 查看当前用户，不带有进程 jobs 查看操作系统当前运行了哪些用户作业 kill 杀死进程 sudo passwd username 设置用户密码 sudo su 切换root用户 ifconfig 查看网卡信息 man 系统参考手册 man n name 在系统手册第n章查看name alias 给命令起别名 alias 别名='指令' 为指令起别名 例如：alias pg='ps aux | grep' 为指令ps aux | grep起别名pg。 umask 指定用户创建文件时的掩码（默认去掉文件的执行权限） 例如：先执行umask 522然后再执行touch file，则file文件的权限为--w-r--r--，数字表示为244。计算方法：777-522=255，文件的权限为--w-r-xr-x，由于系统会去掉执行权限，所以是--w-r--r--。 创建目录，默认大小是4096k。
